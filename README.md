# Sistema de Auditoría y Control de Inventario

Curso: Fundamentos de Programación - Código 213022  
Universidad Nacional Abierta y a Distancia - UNAD  
Fase 5 - Evaluación Final POA  

## Descripción
Programa desarrollado en Python que permite gestionar 
el inventario de una tienda, actualizar el stock de productos 
y generar un informe de pedidos de reabastecimiento.

## Funcionalidades
- Ver lista de productos con stock actual y mínimo requerido
- Actualizar el stock sumando o restando unidades
- Generar informe detallado de productos a reabastecer

## Cómo ejecutar
1. Tener Python instalado
2. Ejecutar el archivo inventario.py
